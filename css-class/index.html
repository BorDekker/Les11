git status

Hiermee check of je een repository hebt of dat je iets moet adden/committen

git init en github

Met git init maak je een repository, dan ga je naar githhub om daar een repository te maken. Dat doe je door op het groene knopje te klikken.

git remote, add en origin

Dan moet je git remote add origin (link), daarme koppel je de repository op je laptop aan github.

Daarna doe je git branch -M (naam), hiermee verander je de naam van je branch.

Dan moet je git push -u origin main, hiermee stuur je de branch van je main naar de branch van Github.

git push, log, add en commit

Dit zijn de basics die je moet kennen om met github te werken.

Zoals git add (naam van bestand) / git add . (add alle bestanden), dat voegt een bestand toe aan een staging area. Zodat je het later kan committen.

Na git push moet je git commit -m (maak een naam b.v.b “first commit”) doen, dan zorg je ervoor dat het git het bestand gaat volgen/bijhouden/tracken. En -m is dat je een commit message toevoegt, bijvoorbeeld: git commit -m "first commit".

Als laatste moet je git push doen, hiermee stuur je alle openstaande commit’s die nog niet gepushed zijn, push je ze allemaal tegelijkertijd door.

Dan heb je nog git log, hiermee kan je checken welke commit je allemaal gedaan hebt wanneer en hoelaat.
